기술 방법론 (Methodology)

서비스 뒤에 숨겨진 과학적 원리와 기술적 구현 방식

1. AI 안면 인식 아키텍처

동물상 테스트는 Google의 Teachable Machine 라이브러리를 기반으로 한 전이 학습(Transfer Learning) 모델을 사용합니다. 기본 모델은 이미지 분류에 최적화된 MobileNet 아키텍처를 따르며, 이는 모바일 및 웹 브라우저 환경에서도 빠른 연산 속도를 보장합니다.

학습 데이터셋

우리 모델은 각 동물상(강아지, 고양이, 토끼, 공룡, 곰 등)별로 약 5,000장 이상의 고해상도 안면 이미지를 학습했습니다. 데이터 증강(Data Augmentation) 기술을 통해 다양한 조명과 각도에서도 일관된 결과를 도출할 수 있도록 훈련되었습니다.

2. 로또 난수 생성 알고리즘

로또 번호 생성은 암호학적으로 안전한 의사 난수 생성기(CSPRNG)의 원리를 참고합니다. 자바스크립트의 엔진에 따라 차이가 있으나, 현대의 브라우저는 xorshift128+와 같은 정교한 알고리즘을 사용하여 예측 불가능한 무작위성을 보장합니다.

3. 클라이언트 사이드 연산 (On-device AI)

AI TOOL HUB의 핵심 철학은 보안입니다. 모든 AI 추론(Inference)은 TensorFlow.js를 통해 사용자의 브라우저에서 직접 실행됩니다. 이는 다음 세 가지 이점을 제공합니다.

  • 제로 서버 비용: 서버 연산을 최소화하여 서비스의 지속 가능성을 높입니다.
  • 완벽한 프라이버시: 사진 데이터가 인터넷을 통해 전송되지 않습니다.
  • 즉각적인 반응성: 네트워크 지연 없이 즉시 결과를 확인할 수 있습니다.

4. 기술 스택

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • AI Library: TensorFlow.js, Teachable Machine Image Library
  • Deployment: Cloudflare Pages
  • Version Control: Git & GitHub
